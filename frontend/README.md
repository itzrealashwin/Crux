# Project Crux - Placement Management System

A comprehensive placement management system for colleges, simplifying the process from job posting to final selection.

## Features

- **Job Management**: Create and manage job opportunities from a single dashboard.
- **Student Profiles**: Build complete student profiles beyond CGPA.
- **Eligibility Checks**: Automatically validate eligibility before application.
- **Application Tracking**: Track applications with complete clarity.
- **Applicant Management**: Review and manage applicants efficiently.

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui (simulated/custom implementation)
- Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
